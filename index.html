Gallery of Projects

Mobile-First

A responsive layout to achieve the different screen sizes, mobile, tablet and desktop. This webpage used HTML and CSS grid box modals for the layout of the thumbnail gallery.

Online Registration Form

Here is a simple web form for users to fill out to collect information. I used HTML elements to create labels to help the user fill the form out correctly. Some of the form fields used were: text input, checkboxes and a select menu that pops down.

Gameshow App

An interactive word guessing game. Building this app is where I learned JavaScript. I had to create a JS function that randomly chooses a letter from the phase word.

Employee Directory

I created this page to have all the employees' contact information along with their photos. This project used API(Application Programming Interface) usage and fetch to select the employee from the list. Then created a JS modal window that displays the employees' contact details.
